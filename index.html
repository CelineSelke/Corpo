About Corpo

Corpo is a text corpus management system designed for the creation, management, and maintenance of text corpora, a type of database that is fundamental to the field of Natural Language Processing. The system includes built-in tools for organizing and searching a corpus according to document metadata and to run linguistic analysis algorithms.
